📖 常见问题解决方案

根据诊断结果，这里是常见问题的解决方案：

❌ 404 Not Found
问题：API 路由返回 404 错误 原因： 1. 相对路径在国际化路由中被错误解析 2. API 路由文件位置不正确 3. Middleware 配置拦截了 API 请求 解决方案： 1. 使用 window.location.origin 拼接完整 URL： const url = `${window.location.origin}/api/creem/create-checkout` 2. 检查文件位置： app/api/creem/create-checkout/route.ts 3. 检查 middleware.ts 配置： matcher: ['/((?!api|...)...)']
⚙️ 环境变量未配置
问题：CREEM_API_KEY 或其他环境变量未配置 解决方案： 1. 在项目根目录创建 .env.local 文件 2. 添加必需的环境变量： CREEM_API_KEY=your-api-key NEXT_PUBLIC_CREEM_PRO_PRODUCT_ID=prod_xxxx 3. 重启开发服务器（Ctrl+C 然后 npm run dev） 4. 再次运行诊断工具验证
🔐 API 认证失败
问题：Creem API 返回 401 Unauthorized 解决方案： 1. 检查 CREEM_API_KEY 是否正确 2. 登录 Creem Dashboard 验证 API Key 3. 确保 API Key 有足够的权限 4. 检查 API Key 是否过期
